# ГенИИ — премиальный лендинг курса

Одностраничный лендинг для курса «ГенИИ» с акцентом на премиальный визуал, мобильный UX и конверсионные блоки. Реализовано на React + Vite + Tailwind CSS с анимациями Framer Motion.

## Что внутри

- Модульная структура `components/`, `sections/`, `data/`
- Мобильный sticky CTA
- Премиальная тёмная эстетика с мягкими градиентами
- Анимации появления и аккуратные hover-эффекты
- Семантичная верстка и фокус на доступность

## Быстрый старт

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Настройка контента

Тексты и данные находятся в `src/data/content.js`.
